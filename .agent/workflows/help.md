# Antigravity コンシェルジュ (The Interactive Guide)

現在のプロジェクト状況（フェーズ）と、あなたの「やりたいこと」に基づいて、最適なコマンドを案内します。

## 1. 状況確認 (Context Check)
まず、現在のフェーズを確認します。
Run: `grep "Current Phase" -A 10 GEMINI.md`

## 2. ヒアリング (Intent Discovery)
あなたの状況に近いものを番号で選んでください。

1. **「新しい機能を作りたい / アイデアを形にしたい」**
   -> `Planning` フェーズへ。 `@[/spec-phase]` が最適です。

2. **「バグを直したい / 実装を進めたい」**
   -> `Execution` フェーズへ。 `@[/impl]` が最適です。

3. **「全体的な設計やDBリングを行いたい」**
   -> `Design` フェーズへ。 `@[/design]` が最適です。

4. **「もう完成した！リリースしていい？」**
   -> `Stable` フェーズへ。 `@[/ship]` で監査を行いましょう。

5. **「コードが汚い / 整理したい」**
   -> `@[/refactor]` で安全に綺麗にしましょう。

6. **「何もわからない / 久々に触る」**
   -> `@[/onboard]` で状況を思い出しましょう。

## 3. コンシェルジュ提案 (The Concierge)

上記の情報とユーザーの入力から、**「今実行すべきコマンド」を1つだけ** 提案してください。

> **💡 AI Suggestion**
> 現在は **[Implementation]** フェーズで、直前の `audit` が失敗しています。
> 
> 推奨コマンド: **`@[/impl]`** (修正を開始)
> (または `@[/ship]` で再監査)

ユーザーが迷っている場合、その推奨コマンドを自動実行してもよいですか？と聞いてください。
