# Antigravity Rules 🪐

**「最適・最速・最高品質」** を実現するための、AIエージェント用・自律開発ルールセット。
既存のプロジェクトや新規プロジェクトにこの設定を読み込ませるだけで、開発プロセスが劇的に効率化されます。

## 📦 ダウンロード (Download)

以下のリンクから配布用キット (`.zip`) をダウンロードできます。

**[👉 AntigravityKit.zip をダウンロード](https://github.com/JyozukaTukasa/AntigravityRules/raw/main/AntigravityKit.zip)**

---

## 🚀 使い方 (Usage)

1.  **ダウンロード**: 上記の Zip ファイルをダウンロードします。
2.  **展開**: あなたのプロジェクトの **ルートディレクトリ** に展開してください。
    - `.agent/`: ワークフロー定義
    - `.github/`: PRテンプレート
    - `ANTIGRAVITY_MANUAL.md`: 運用マニュアル
    - `GEMINI.md`: プロジェクト定義テンプレート
3.  **開始**:
    - AIエージェントに「`@[/onboard]` して」と話しかけるだけで、Antigravity モードが起動します。

## 🛠 含まれる機能

- **`@[/ship]`**: テスト・セキュリティ・品質・パフォーマンスをワンクリックで全監査。
- **`@[/consult]`**: プロジェクトの改善案をプロアクティブに提案。
- **`@[/sync]`**: ドキュメントとコードの乖離を自動検知・修正。
- **その他**: `@[/design]`, `@[/impl]`, `@[/refactor]` などのタスク特化型ワークフロー。
