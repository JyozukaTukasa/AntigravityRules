## 💡 Antigravityの特徴

### 1. 「考える」を代行する (Spec Phase)
`@[/spec-phase]` で、あなたの曖昧なアイデアをAIがプロの仕様書に変換します。

### 2. 「守り」を自動化する (Defense Layers)
`@[/config]` でセキュリティレベルを設定可能。スタートアップ(L1)から国家防衛(L4)まで、プロジェクトごとの最適解で守ります。

### 3. 「速さ」を追求する (Deep Optimization)
開発中はSmart Testingで数秒でテスト完了。出荷時は全監査。速度と安全性のいいとこ取りを実現しました。
